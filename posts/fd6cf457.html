Java中的注释

  1. 单行注释 //
  2. 多行注释 /* */
  3. 文档注释 /** */

关键字

标识符

  1. 所有标识符都应该以字母、$、下划线 ( _ ) 开始.
  2. 首字符之后可以是字母、$、下划线或数字的任何字符组合
  3. 不能使用关键字作为变量名或方法名
  4. 标识符大小写敏感

2.数据类型

强类型语言

  • 所有变量都必须先定义后才能使用(安全性高)

弱类型语言

Java的数据类型

  • 基本类型

    1. 数值类型

      • 整数类型

        – byte(一个字节)：范围：-128~127

        –short(两个字节)：范围：-2^15~(2^15-1)

        –int(四个字节)：范围:-2^31~(2^31-1)

        –long(八个字节)：范围：-2^63~(2^63-1)

      • 浮点类型

        –float(四个字节)

        –double(八个字节)

      • 字符类型

        –char(两个字节) 本质还是数字

    2. boolean类型(占一位) true / false

  • 引用类型

    1. 类
    2. 接口
    3. 数组

什么是字节

  • 位(bit):计算机中内部数据存储的最小单位，11011100是一个八位二进制数

  • 字节(byte):计算机中数据处理的基本单。

  • 1B(byte,字节)=8bit(位)。

  • 字符：计算机中使用的单个字母、数字、字和符号。

    转换：1B=8bit;

​ 1024B=1KB

​ 1024KB=1M

​ 1024M=1G

3.类型转换

低————————————————————–高

byte,short,char —>int—>long—>float—>double

  • 运算中，不同类型的数据先转化为同一类型，再进行运算

  • 强制类型转换

    int number=20;
byte num=(byte)number;

    注 ：

    1. 不能对布尔值进行转换
    2. 不能把对象类型转换为不相干的类型
    3. 在把高容量转换到低容量时，用强制转换
    4. 转换时可能存在内存溢出，或者精度问题

  • 自动类型转换

    低容量–>高容量

    int nt=10;
double num=nt;

4.变量、常量

变量

  • 定义：可以变化的量

  • java是强类型语言，变量都必须声明其类型

  • 要素：变量名，变量类型和作用域

    注意事项：

    • 每个变量都有类型，类型可以是基本数据类型，也可是引用数据类型。
    • 变量名符合标识符的命名规范
    • 声明变量最后以分号结尾

变量作用域

  • 局部变量

    • 必须声明和初始化值

  • 实例变量

    • 从属于对象，若不进行初始化，默认为此类型的默认值
      • boolean 布尔值 默认 false
      • 除了基本数据类型，其余默认值都为null

  • 类变量 static修饰的变量

    • 从属于类，随类出现和消失

变量的命名规范

  • 所有变量、方法、类名：见名知意
  • 类成员变量/局部变量/方法名：驼峰命名法 首字母小写：monthSalary
  • 常量：大写字母和下划线：MAX_VALUE
  • 类名：首字母大写驼峰原则：MainTex

常量

  • 常量：初始化后不能再改变值

     final 常量名=值;
 final double PI=3.14;

  • 常量名一般使用大写字符

5.运算符

  • 算术运算符：+、-、*、/、%、++、–

    • 自增 ++

      a++ 与 ++a 等价于 a=a+1;
区别：a++ 先赋值后运算
int a=4;

int b;
int c;
b=a++;
c=++a;
System.out.print(b);//4
System.out.print(c);//6
++a 先运算后赋值
注：这里的运算是当前值的自增或自减运算

    • 自减 - - 与自增原理相同则不再赘述

    • 字符串连接符： + ，String 拼接

  • 赋值运算符：=，+=，-=，*=，/=

  • 关系运算符：>、<、>=、<=,==,!= ,instanceof

  • 逻辑运算符：&，|，&&，||，^, ~, >>，<< , >>>

    短路运算：

    • && 逻辑与：两个变量都为真，结果才为true
    • || 逻辑或：两个变量有一个为真，则结果为真

    位运算(二进制运算)

    • | 数值每一位做或运算

    • & 数值每一位做与运算

    • ^ 异或运算 同为0 异为1

    • ~ 非运算 取反

      示例：
   A=0011 1100;
   B=0000 1011;
----------------
A&B= 0000 1000;
A|B= 0011 1111;
A^B= 0011 0111;
~B = 1111 0100;

    • << 左移 *2

    • ,>> 右移 /2

  • 条件运算符 ？：（三元运算符）

    X？y:z;
如果x为真;则结果为 y,否则结果为 z;
int source=60;
String ty=source<60?"不及格":"及格";
System.out.print(ty);

    优先级 加（）

6.包机制、JavaDoc

包机制

  • 为了更好组织类，Java提供了包机制，用于区别类名的命名空间

  • 一般使用公司域名倒置作为包名

  • 若要使用其他包中的成员，需要使用 ”import“ 进行导包，完成引用

    import 包名.类名;
import 包名.*;导入此包下所有的类；

JavaDoc

  • javadoc命令是用来生成自己API文档的
  • 参数信息
    • @author 作者名
    • @version 版本号
    • @since 知名需要最早使用的jdk版本
    • @param 参数名
    • @return 返回值情况
    • @throws 异常抛出情况

通过命令行产生javaDoc文档：javadoc 参数 文件名.java

​ - 参数：-encoding UTF-8 -charset UTF-8 Doc.java
